Activity 5.2.3. Practice with Set Operations.

Let \(A=\{1, 2, 3, 4, 5, 6\}\) and \(B=\{2, 4, 6, 8, 10, 12\}\) and the Universal set \(U=\{n\in\mathbb{Z}: 1\leq n\leq 12\}\text{.}\)
(a)
Find \(A\cup B\text{.}\)
(b)
Find \(A\cap B\text{.}\)
(c)
Find \(A- B\text{.}\)
(d)
Find \(B- A\text{.}\)
(e)
Find \(B^{C}\text{.}\)
